# JavaScript 合集

##### 食用方式 : 通过在 `<head>` 中添加 
`<script type="text/javascript" src="https://moyu-official.github.io/JavaScript/*.js"></script>`

##### 合集内容一览

在要显示日期和时间的地方添加 `<div id="display-date-and-time"></div>` , 也可使用 `<h1>` `<p>` 等标签
